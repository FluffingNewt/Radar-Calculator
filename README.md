# radar-range-equation

The **Radar Range Equation** is used to calculate the theoretical **range of the target**.

<div style="text-align: center;">

$\displaystyle
R_{max} =
\frac{P_t \text{ } G_t \text{ } G_r \text{ } \lambda^2  \text{ } \sigma}
{(4\pi)^3 \text{ } R^4}
$

</div>
